# Medical_Chatbot

# 🏥 Medical Chatbot using Encyclopedia Knowledge Base

## 📌 Overview

This project is an AI-powered medical chatbot that provides information about diseases, symptoms, and treatments using a structured medical encyclopedia dataset. It is designed to assist users with preliminary medical knowledge and awareness.

---

## 🚀 Features

- 🔍 Symptom-based disease suggestions
- 📖 Encyclopedia-based medical responses
- 💬 Natural language interaction
- ⚡ Fast and accurate query handling
- 🧠 AI/NLP-powered response generation

---

## 🛠️ Tech Stack

- Language: Python
- Framework: Flask
- NLP: Transformers
- Database: FAISS
- Frontend: Streamlit

---

## 📂 Dataset

The chatbot uses a medical encyclopedia dataset containing structured information about:

- Diseases
- Symptoms
- Causes
- Treatments

---

## 🧪 How It Works

1. User inputs a query (symptoms or disease)
2. NLP model processes the input
3. Relevant information is retrieved from the dataset
4. Chatbot generates a response

---

## 🎯 Future Improvements

- Integration with real-time medical APIs
